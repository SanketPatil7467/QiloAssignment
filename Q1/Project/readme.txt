To perform operations like create blog, read blog, delete blog, modify blog, add comments and to like and dislike the blog posts, 3 databases are created and the communication between this databases and server is performed using fastapi.
Initial Page:
1. All the blogs from all bloggers are shown to any user on web portal and user have access to like/dislike the post and comment on the posts.
2. Register option is provided to user's to publish their own bloggers
3. After successful Registration and login user will redirected to its own portfolio where all blog posts are which are published by him/her will be shown
4. User have options to modify content, add blogs and delete blog from database.
